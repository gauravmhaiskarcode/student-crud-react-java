# Student Management System (React + Spring Boot)

This is a full-stack **Student Management CRUD** application built with:

- **React.js** (Frontend)
- **Spring Boot** (Backend)
- **MySQL** (Database)

It allows users to **Create, Read, Update, and Delete** student records with a responsive UI and RESTful API.

---

## 🔧 Technologies Used

### 🖥️ Frontend
- React JS
- Axios
- Bootstrap 5
- React Router DOM

### 🧠 Backend
- Spring Boot
- Spring Data JPA
- Lombok
- MySQL
- Java 17+

---

## 📸 Features

- Add new student
- View all students
- Edit student details
- Delete student
- Form validations
